# Manifold Markets Skill for Claude Code

A Claude Code plugin that teaches Claude how to interact with the Manifold Markets prediction market API.

## Installation

```bash
/plugin marketplace add Thomas-Lemoine/manifold-markets-skill
```

## What's Included

This skill provides Claude with knowledge about:
- REST API endpoints and authentication
- WebSocket real-time updates
- Placing bets, limit orders, and multi-outcome bets
- Market creation and resolution
- Portfolio analysis and position tracking
- AMM math and orderbook simulation
- Bulk data access via Supabase

## Usage

Once installed, Claude will automatically use this skill when you ask about Manifold Markets tasks like:
- Fetching market data
- Placing or analyzing bets
- Building trading bots
- Working with the Manifold API
